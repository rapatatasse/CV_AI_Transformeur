prgramme qui transforme les cv et base de donner par l'ia en local.
Utilisation de PowerShell pour install des logiciel et lancement
Utilisation de Python pour transformer le pdf en txt et les envoyer à LM Studio pour remplir une base de donnée
Utilisation de LM studio pour trabailler avec IA en local transformant avec le bon model les txt en csv 
Utilisation de Node JS pour afficher le CSV dans site HTML et pourvoir filtrer les CV.
